Lab 2: ASCII Forest

-----------
DESCRIPTION

In this lab, the user will input two numbers, one for the number of trees and one for the size. Then Lab3.asm will print these trees in ASCII to the console window.

-----------
FILES

-
Diagram.pdf

This file includes a block diagram of the program design. The application draw.io was used to draft the diagram and generate the pdf.

-
Lab3.asm

This file includes the assembly code of the lab. It handles the actual project. Handles integers of any size, but only a multiple > 0 and size >1 can be printed.
prints trees in ASCII based of user input for number and size

-----------
INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and Runtime Simulator (MARS). Enter the test case as a program argument and run using MARS.
